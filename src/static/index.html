Positive Vibes

Vibes positivas, mensagens, músicas, imagens para você se lembrar de que o mundo é muito belo, e sua existência é um universo.

Recursos

Frases

O recurso de frases é construido em cima de raspagem de dados do site O Pensador.

/quotes

Retorna um JSON contento todas as frases.

/quotes/random

Retorna um JSON contendo uma frase aleatória.

Músicas

O recurso de músicas utiliza a API do Jamendo para consumir dados e arquivos de playlists específicas do Jamendo.

/songs

Retorna um JSON contendo informações de todas as músicas disponíveis.

/songs/random

Retorna um arquivo de audio MP3 aleatório.

Imagens

O recurso de imagens utiliza a API do Pixabay para consumir dados e arquivos do Pixabay.

/images

Retorna um JSON contendo todas as imagens.

/images/random

Retorna um arquivo de imagem aleatório.

Podcast

O recurso de podcasts utiliza o feed do podcasts Acenda sua Luz e disponibiliza seus episódios

/podcasts

Retorna um JSON contendo todos os episódios do Podcast Acenda sua Luz.

/podcasts/random

Retorna JSON com um episódio.

Referências

O Pensador

As frases providas pela API são obtidas através de uma raspagem de dados realizada no site. O Pensador

Jamendo

As músicas são obtidas por meio da API do Jamendo, é necessário se cadastrar com uma conta de desenvolvedor para obter uma chave de acesso à API.

Pixabay

As imagens são obtidas por meio da API do Pixabay e assim como no caso do Jamendo, é necessário se cadastrar com uma conta de desenvolvedor para obter uma chave de acesso à API.

Podcast

O podcast Acenda sua Luz é produzido e apresentado pela colunista e escritora Carol Rache, apesar de ser relativamente recente, é um dos meus podcasts favoritos pois o mesmo contém muitas reflexões e ensinamentos que eu carrego comigo no meu dia-a-dia.
